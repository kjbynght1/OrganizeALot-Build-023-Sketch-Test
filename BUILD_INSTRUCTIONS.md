# Build 023 Android package

This project is ready for Android Studio or GitHub Actions.

## Fastest build with GitHub Actions
1. Upload the project contents to a GitHub repository.
2. Open the repository's **Actions** tab.
3. Run **Build OrganizeALot Android APK**.
4. Download the **OrganizeALot-Build023-APK** artifact when the run completes.

## What to test on the Samsung phone
1. Start an inspection and enter an Inspection ID and Address.
2. Take a photo and tap **Use Photo**.
3. Confirm the photo appears under the correct inspection item.
4. Open Samsung Gallery and confirm the copy appears under:
   `Pictures / OrganizeALot / <Inspection ID>`
5. Reopen the inspection and confirm the photo and Delete button remain under the correct item.
6. Delete the photo in OrganizeALot and confirm the Gallery backup remains.
